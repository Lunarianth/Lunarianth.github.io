# python和java的差别
1. 语法风格
Python：语法简洁，接近自然语言，强调"少写代码"。比如缩进就是语法的一部分。  
Java：语法更严格，冗长，需要写很多结构（类、类型声明）

```python
print("Hello")
```

```java
public class Main {
    public static void main(String[] args) {
        System.out.println("Hello");
    }
}
```

2. 类型系统
python: 动态类型(变量类型运行时决定)
Java：静态类型(必须提前声明类型)  

3. 执行方式
Python:解释型语言(逐行执行)  
Java:编译+运行在JVM上
Java通常在性能上更稳定，更快(尤其是长期运行程序)

## Python的垃圾回收机制
每个对象都有一个引用计数:
```python
a=[]
b=a      #引用+1
del a    #引用-1
```
当引用计数变成0-> 立即释放内存

## Java的垃圾回收机制
交给JVM
